## 챕터 1: 코워크 모드 스킬 (Claude Code Skills)

### 1.1 개요
Claude Code 스킬 시스템은 `.skills/` 디렉토리에 `SKILL.md` 파일로 정의된다.
각 스킬은 독립 트리거·역할·출력 형식을 가지며, 코워크 모드에서 자동 라우팅된다.

### 1.2 등록된 스킬 목록 (5종)

| 스킬명 | 위치 | 수준★ | 역할 | 트리거 키워드 |
|---|---|---|---|---|
| corp-financial-analysis | .skills/corp-financial-analysis/ | ★★★★★ | 6트랙 재무·세무 분석 | 법인세 절세, 재무 분석 |
| pptx | .skills/pptx/ | ★★★★ | 네이비 표준 PPT 생성 | PPT 만들어, 슬라이드 |
| docx | .skills/docx/ | ★★★★ | 표준 Word 보고서 생성 | Word 보고서, docx |
| xlsx | .skills/xlsx/ | ★★★★ | 시뮬레이터·분석 Excel | Excel 만들어, 시뮬 |
| pdf | .skills/pdf/ | ★★★ | 최종 배포용 PDF 변환 | PDF 변환, 인쇄용 |

### 1.3 스킬 트리거 메커니즘
1. 사용자 자연어 입력 → 트리거 키워드 매칭
2. 매칭 스킬 SKILL.md 참조 → 역할·출력형식 확인
3. 해당 스킬 실행 (파일 생성 or 분석 수행)
