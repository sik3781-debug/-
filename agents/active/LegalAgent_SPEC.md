# LegalAgent — 표준 명세서 (SPEC)

## 1. 단일책임
법인 운영 현황 입력 → 상법 절차 위반·특수관계인 거래 공정거래법·차명주식 형사 리스크 진단 및 해소 방안 산출.

**필수**: 기업개요(brief), 특수관계인 거래 현황, 명의신탁 여부

**선택**: 이사회 의사록, 정관 내용
