# 고객사컨텍스트Agent (client_context_loader)

## 1. 단일책임
고객사명 또는 상담ID 입력 → 이전 분석·이슈·진행상황을 자동 요약하여 모든 컨설팅 작업의 진입 컨텍스트 제공.

## 2. 입력

### 필수
| 항목 | 형식 | 예시 |
|---|---|---|
| 고객사명 또는 상담ID | 문자열 | "(주)경남제조" 또는 "C-2024-047" |

### 선택
- 조회기간 (기본: 최근 12개월)
- 조회 분야 필터 (가지급금 / 승계 / 주식평가 / 전체)

## 3. 출력

요약 3페이지 이내 (Markdown):

```markdown
## 고객사 컨텍스트 요약
**고객사**: (주)경남제조  **상담ID**: C-2024-047  **최종갱신**: 2026-05-01

### 기본 정보
- 업종: 금속가공 (KSIC 25120) / 자본금 5억 / 임직원 45명
- 대표이사: 홍길동 (60세) / 지분 70%
- 신용등급: BB+ (추정, 2025-11)

### 이전 컨설팅 이력
| 일자 | 유형 | 결과 | 산출물 |
|---|---|---|---|
| 2024-08-15 | 가지급금 해소 | 완료 | 가지급금_해소_시뮬레이션.xlsx |
| 2025-03-20 | 비상장주식 평가 | 진행중 | 주식평가보고서_초안.pptx |

### 미해결 이슈
1. 가업승계 시나리오 미결정 (자녀 2명, 지분 분산 문제)
2. 임원퇴직금 규정 미비 (세무조사 리스크)

### 진행 중 작업
- 비상장주식 이동 전략 검토 중 (담당: DataValidationAgent)

### 다음 제안
1. 가업승계 시뮬레이션 실행 (우선순위 高)
2. 임원퇴직금 규정 신설 (세무조사 대비)
```

**필수 필드**: 기본정보 / 이전컨설팅 / 미해결이슈 / 진행중작업 / 다음제안

## 4. 사용 도구
| 도구 | 용도 |
|---|---|
| memory/고객사/ | 고객사별 디렉토리 자동 검색 |
| Semantic Search | 키워드 기반 의미 검색 (고객사명 오타·약칭 허용) |
| 내부 CRM 연동 | 상담 이력 조회 (구현 시 연동) |

**모델:** claude-haiku-4-5-20251001 (빠른 요약 위주 — 비용 최적화)

## 5. 자가검증 3축

### ① 정보 신선도
- 90일 이내 정보: 우선 표시
- 90~365일: 날짜 표시 후 포함
- 365일 초과: "오래된 정보" 경고 태그

### ② 누락 없음
- 미해결 이슈 0건 누락 (전수 표시)
- 진행 중 작업 상태 최신화 여부 확인

### ③ 4자관점 적용
- 이전 컨설팅 내 4자관점 커버리지 표시
- 미커버된 관점 "다음 제안"에 반드시 포함

## 6. 협력 에이전트
- **상류**: 없음 (모든 컨설팅 작업의 **첫 단계**로 자동 호출)
- **하류**: 전체 컨설팅 에이전트 (진입 컨텍스트로 패스스루)

**자동 호출 조건**: 고객사명 또는 상담ID가 포함된 모든 요청 시 사전 실행

## 7. 실패 처리
| 상황 | 처리 |
|---|---|
| 고객사 미등록 | "신규 고객사 — 기본 정보 입력 필요" + 입력 양식 제공 |
| 상담ID 불일치 | 유사 ID 목록 3개 제안 |
| memory 디렉토리 비어 있음 | 기본 정보만 표시 + "이력 없음" 명시 |

## 8. KPI
| 지표 | 목표 |
|---|---|
| 호출 응답시간 | 5초 이내 |
| 평균 토큰 | 1,500 토큰 이내 |
| 정보 누락률 | 0% (등록 이슈 전수) |
