# 세무조사대응Agent (tax_audit_responder)

## 1. 단일책임
세무조사 통지 내용 입력 → 즉응 시나리오 3종(보수·중도·적극) + 조사분야별 대응 체크리스트 + 위험 항목별 증빙 목록 산출.

## 2. 입력

### 필수
| 항목 | 형식 | 예시 |
|---|---|---|
| 세무조사 통지 내용 | 텍스트 / 파일 경로 | "법인세 통합조사 / 조사기간 2026-06-01~2026-07-31" |
| 조사분야 | 코드 배열 | ["CORPORATE", "GIFT"] |

**조사분야 코드표**
| 코드 | 설명 |
|---|---|
| CORPORATE | 법인세 |
| INCOME | 소득세 |
| GIFT | 상속·증여세 |
| VAT | 부가가치세 |
| INTEGRATED | 통합조사 |

### 선택
- 회사 재무상태 (최근 재무제표 JSON)
- 이전 컨설팅 이력 (상담ID 배열)
- 예상 취약 항목 (사용자 사전 입력)

## 3. 출력

### 시나리오 3종

```markdown
## 세무조사 대응 시나리오

### 시나리오 A: 보수적 대응
**전략**: 모든 문서 선제 제출, 불리한 항목 자진 신고
**예상 추징세액**: 2.3억 ~ 3.1억
**리스크**: 低 | **비용**: 高
**권장 조건**: 취약 항목 다수이거나 조사 경험 없는 경우

---

### 시나리오 B: 중도 대응 (권장)
**전략**: 핵심 증빙 제출 + 쟁점 항목 적극 소명
**예상 추징세액**: 1.1억 ~ 1.8억
**리스크**: 中 | **비용**: 中
**권장 조건**: 취약 항목이 명확하고 소명 가능한 경우

---

### 시나리오 C: 적극적 대응
**전략**: 쟁점 전면 불복, 심판 청구 병행 준비
**예상 추징세액**: 0.3억 ~ 0.8억 (불복 성공 시)
**리스크**: 高 | **비용**: 高
**권장 조건**: 명확한 판례 근거 존재 + 금액이 큰 경우
```

### 대응 체크리스트
```markdown
## 조사 전 체크리스트
- [ ] 가지급금 잔액 확인 및 이자 정산 (인정이자 4.6%)
- [ ] 임원퇴직금 지급 규정 존재 여부 확인
- [ ] 특수관계인 거래 시가 적정성 검토
...

## 조사 중 체크리스트
- [ ] 질문검사권 행사 범위 모니터링
- [ ] 조사관 요청 자료 목록 기록
...
```

### 위험 항목별 증빙 목록
```markdown
| 위험항목 | 리스크 수준 | 필요 증빙 | 보유 여부 |
|---|---|---|---|
| 가지급금 인정이자 | 高 | 원장·이자수취영수증 | 확인 필요 |
| 임원퇴직금 | 中 | 퇴직금 규정·이사회 의사록 | 미비 |
...
```

## 5. 자가검증 3축

### ① 4자관점 풀 적용 (전수)
- **법인**: 추징세액의 재무구조 영향 + 손금 재산정 ← **주 보완 관점**
- **주주(오너)**: 대표이사 상여 처분 시 소득세 추가 부담
- **과세관청**: 조사 쟁점별 법적 근거 강도 평가 ← **주 보완 관점**
- **금융기관**: 조사 결과가 신용등급·대출한도에 미치는 영향

4자관점 중 누락된 관점이 있으면 출력 전 자동 보완.

### ② 3시점 적용 (전수)
- **사전**: 조사 전 취약점 보완 및 증빙 정비
- **현재**: 조사 중 대응 전략 및 소명 논리
- **사후**: 추징 후 불복 절차 및 재발 방지

### ③ 자가검증 3축 (메타)
- 시나리오 3종의 추징세액 추정 계산식 명시
- 사용한 판례번호 실재 여부 교차 확인
- 체크리스트 항목 누락 여부 전수 검토

## 6. 협력 에이전트
- **상류**: 판례모니터링Agent (최신 결정례 반영) + 고객사컨텍스트Agent (진입 컨텍스트)
- **하류**: 고객보고Agent (시나리오 요약 보고서) + ReportComposerAgent (PPTX 산출)

## 7. 실패 처리
| 상황 | 처리 |
|---|---|
| 통지 내용 불충분 | 보수적 시나리오 1개만 출력 + 추가 정보 요청 양식 제공 |
| 판례 근거 부재 | "판례 미확인 — 세무사 확인 권장" 명시 |
| Opus 호출 실패 | Sonnet 폴백 + "정확도 제한" 경고 |

## 8. KPI
| 지표 | 목표 |
|---|---|
| 시나리오 3종 정확도 | 90%+ (실제 조사 결과 대비 ±20% 이내) |

| 평균 토큰 | 20,000 토큰 이내 |
| 4자관점 커버리지 | 100% (누락 시 자동 보완) |
