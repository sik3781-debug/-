# LegalRiskHedgeAgent — `/법률리스크체크`

계약·정관·이사회·주총·등기 5대 영역 리스크 점검 + 정량화 + 헷지 매핑.

## 핵심 기능
- 13종 룰북 일괄 점검 (특수관계 거래·이사 자기거래·경업금지·민법§103·§104·형법§356·정관 위반·이사회 정족수·주총 통지·등기 지연·자본시장법§178·영업비밀 침해·부당과소신고)
- 트리거된 룰별 즉시 헷지 매핑
- 위험도 우선순위 정렬 (CRITICAL → HIGH → MEDIUM → LOW)
- 형사·민사·행정 처분 분류
- 위약금·손해배상·과태료·가산세 정량화

## 핵심 법령
- 상법 §289~§542 (회사 운영 — 정관·이사회·주총·등기)
  - §382의3 (이사 충실의무), §397 (경업금지), §398 (자기거래)
  - §391~§393 (이사회 의사록), §433~§435 (주총 결의)
  - §37 (변경등기 2주 이내)
- 민법 §103 (반사회질서 — 무효), §104 (불공정 — 무효)
- 형법 §356 (업무상 배임 — 5년/1500만)
- 자본시장법 §178 (부정거래 — 5년/5억)
- 부정경쟁방지법 §18 (영업비밀 침해 — 10년/5억)
- 국기 §47의3 (부당과소신고 가산세 40%)

## 5축 리스크 검증
| 축 | 검증 항목 |
|---|---|
| DOMAIN | 5대 영역(계약·정관·이사회·주총·등기) 모두 점검 |
| LEGAL | 검출 룰 모두 법령 인용 |
| CALC | 금전 위험액 합계 정합 |
| LOGIC | 위험도 분포 ↔ 검출 룰 수 일치 |
| CROSS | 4자관점 본문 노출 |

## 4단계 헷지
- **사전**: 정관 + 이사회결의 적법성 검토 / 정상가 입증자료 확보
- **진행**: 계약서 조문별 리스크 표시 + 의사록 기명날인 + 등기 신청
- **사후**: 이행 모니터링 + 의사록 10년 보존 + 정기 검토 캘린더
- **최악**: 의사록·결의서 백업 + 정상가 입증자료 + 형사 변호사 자문

## 후속 세션 심화 권고
- 계약서·정관 텍스트 NLP 기반 리스크 조항 자동 하이라이트
- 판례 검색 자동 통합 (LAW_API_ID 발급 후)
- 형법§356 양형 가중·감경 사유 알림
- 합의·자율 신고 감면 절차 안내
