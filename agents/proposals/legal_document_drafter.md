# LegalDocumentDrafterAgent — `/법무서류`

정관·이사회의사록·주총의사록·동의서·확인서 + 자기주식취득결의서·합병결의서 (총 7종) 표준 양식 자동 작성.

## 핵심 기능
- 7종 표준 양식 (정관/이사회/주총/동의서/확인서/자기주식취득/합병)
- 인지세 자동 산출 (인지세법§3 별표 — 금액 구간별)
- 등록면허세 자동 산출 (지방세법§28)
- 공증 절차 안내 + 예상 공증료 (공증인법§30)
- 결의→의사록→등기→공증 시퀀스 정합 점검
- 누락 필드 자동 진단 + 안내

## 핵심 법령
- 상법 §289~§542
  - §289 (정관 절대적 기재사항)
  - §312 (이사 선임), §391 (이사회 결의), §393 (의사록)
  - §363 (주총 소집통지 2주), §368 (보통결의), §434 (특별결의)
  - §341 (자기주식 취득), §462의3 (배당가능이익)
  - §522 (합병계약), §527의5 (채권자 이의), §530 (합병등기)
  - §37 (변경등기 2주), §396 (의사록 본점 10년 보존)
- 부동산등기법 §29 (등기 신청 요건)
- 공증인법 §30 (공정증서)
- 인지세법 §3 + 별표 (인지세 세율)
- 지방세법 §28 (등록면허세)

## 5축 리스크 검증
| 축 | 검증 항목 |
|---|---|
| DOMAIN | 7대 양식 중 하나에 해당 |
| LEGAL | 양식별 [근거: 상법§...] 본문 포함 |
| CALC | 인지세 + 등록면허세 합계 정합 |
| LOGIC | 시퀀스 단계 정의 |
| CROSS | 4자관점(법인·주주·과세관청·금융기관) 본문 |

## 4단계 헷지
- **사전**: 정관·결의 사전 정비 + 당사자 신원 확인
- **진행**: 5축 검증 + 기명날인 위치 명시 + 등기 2주 이내
- **사후**: 등기·공증 완료 확인 + 본점 10년 보존 + 디지털 백업
- **최악**: 등기 거부 시 즉시 보정 + 인지 누락 시 자진납부 50% 감면

## 후속 세션 심화 권고
- 표준 양식 7종 → 25종 확장 (분할·해산·우선주·전환사채 등)
- 등기 신청서 자체 자동 생성
- 신주발행 등록면허세 전환사채·신종자본증권 차이 반영
- 부동산등기법§29 양식 본문 자동 인용
