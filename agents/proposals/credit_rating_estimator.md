# 신용등급추정Agent (credit_rating_estimator)

## 1. 단일책임
재무제표 3년 + 업종 + 자본금 입력 → 추정 신용등급(AAA~D) 산출 및 정책자금 매칭 5종 제시.

## 2. 입력

### 필수
| 항목 | 형식 | 예시 |
|---|---|---|
| 재무제표 3년 | JSON 또는 Excel 경로 | {2022, 2023, 2024} |
| 업종코드 | KSIC 5자리 | 25120 |
| 자본금 | 원 단위 정수 | 500000000 |

### 선택
- 업력 (연수)
- 임직원 수
- 정책자금 이력 (기관명·금액·잔액)

## 3. 출력

```json
{
  "추정등급": "BBB+",
  "산출근거": "부채비율 120%, 이자보상비율 3.2, EBITDA 마진 12.5%",
  "부채비율": 1.20,
  "이자보상비율": 3.2,
  "EBITDA": 350000000,
  "동종업종_평균등급": "BB+",
  "정책자금매칭5종": [
    {"기관": "중진공", "상품": "신성장기반자금", "한도": "10억", "금리": "2.5%"},
    {"기관": "기보", "상품": "기술보증", "한도": "5억", "보증비율": "85%"},
    ...
  ]
}
```

## 4. 사용 도구
| 도구 | 용도 |
|---|---|
| ECOS_API | 가중평균차입이자율 (한국은행 경제통계시스템) |
| DART_API | 동종업종 비교 (금융감독원 전자공시) |
| 내부 정책자금 DB | 기관별 지원 요건·금리 매칭 |

**모델:** claude-sonnet-4-6

## 5. 자가검증 3축

### ① 계산 정확성
- 부채비율 = 부채총계 / 자본총계 × 100
- 이자보상비율 = 영업이익 / 이자비용
- EBITDA = 영업이익 + 감가상각비 + 무형자산상각비
- 3개 연도 평균값 사용 시 가중치 명시 (최근년도 40%, 전년도 35%, 전전년도 25%)

### ② 법령·통계 근거
- 동종업종 평균: DART 동일 KSIC 코드 기업 최근 3개년 평균
- 한국은행 기준금리 반영 (ECOS_API 실시간)
- 신용등급 기준: KIS·NICE 공개 기준표 준용

### ③ 4자관점
- **법인**: 신용등급이 법인세 손금(이자비용) 및 재무구조에 미치는 영향
- **주주(오너)**: 개인 연대보증 여부·보증한도 변화
- **과세관청**: 정책자금 수령 시 익금산입 여부
- **금융기관**: 담보인정비율·대출한도·금리우대 조건 변화 ← **주 보완 관점**

## 6. 협력 에이전트
- **상류**: DataValidationAgent (재무제표 수치 정합성 검증)
- **하류**: 정책자금매칭Agent (매칭 5종 상세 실행)

## 7. 실패 처리
| 상황 | 처리 |
|---|---|
| 재무 데이터 1개년 부족 | 보수적 추정(BB) 산출 + "데이터 보완 필요" 경고 |
| ECOS/DART API 타임아웃 | 전월 캐시 데이터 사용 + 날짜 명시 |
| 3회 연속 오류 | AutoFixAgent 에스컬레이션 |

## 8. KPI
| 지표 | 목표 |
|---|---|
| 등급 정확도 | 80%+ (실제 금융기관 등급 대비 ±1 등급 이내) |
| 응답시간 | 30초 이내 |
| 평균 토큰 | 5,000 토큰 이내 |
