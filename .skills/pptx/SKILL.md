# pptx

## 역할
중기이코노미 네이비 표준 디자인 시스템으로 PPT 산출물을 생성하는 스킬.
pptxgenjs 기반, hex # 접두사 없는 색상 코드 사용.

## 트리거
- "PPT 만들어", "슬라이드 생성", "pptx 출력"
- "보고서 PPT", "컨설팅 발표자료"

## 디자인 시스템

| 요소 | 값 |
|---|---|
| 메인 컬러 | 000080 (네이비) |
| 배경 | FFFFFF |
| 구분선 | D1D9E6 |
| 액센트 | 1E40AF |
| 세무·절세 | B45309 |
| 재무·리스크 | 0D9488 |
| 승계·주식 | 9F1239 |
| 폰트 (한글) | 나눔고딕 |
| 폰트 (수치) | Calibri |
| 제목 | 20~24pt Bold |
| 본문 | 9~14pt |

## 슬라이드 구조 원칙
- 콘텐츠: 흰색 배경 + 상단 네이비 헤더바 (h=0.72")
- 좌측 액센트바 (w=0.06~0.08")
- shadow: makeShadow() 매번 새 객체 생성 (재사용 금지)
- 표제부: FFFFFF 배경, 상단 네이비 극세 라인 (h=0.06")
- 엔딩: "감사합니다" 46pt Bold 네이비, 연락처 하단

## 금지사항
- hex # 접두사 사용
- shadow 객체 재사용
- 골드 장식선
- 텍스트 전용 슬라이드 (KPI·표 혼용 권장)
- "2025년 귀속 세법 기준" 문구
- "경력 약 15년" 문구
