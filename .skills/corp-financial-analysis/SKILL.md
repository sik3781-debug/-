# corp-financial-analysis

## 역할
중소기업 법인 재무·세무·승계 종합 컨설팅 분석 스킬.
재무제표 기반 6트랙 병렬 분석 → 절세·리스크·의사결정 종합 보고서 산출.

## 트리거
- "법인세 절세", "법인 재무 분석", "세금 줄이기"
- "가지급금", "비상장주식", "가업승계", "임원 급여"
- "4자관점 분석", "재무구조 진단"

## 6트랙 분석 프레임워크

| 트랙 | 내용 | 산출 |
|---|---|---|
| T1 세무분석 | 법인세·부가세·원천세 리스크 | 절세 여지 금액 |
| T2 손금최적화 | 손금 누락·과대 계상 점검 | 추가 손금 항목 |
| T3 주주분석 | 배당·급여·퇴직금 최적 Mix | Mix 시뮬레이션 |
| T4 승계분석 | 비상장주식 평가·이동 경로 | 절세 시나리오 |
| T5 리스크헷지 | 세무조사·법인자금 리스크 | 리스크 등급 |
| T6 재무구조 | 신용등급·가지급금·차입금 | 개선 로드맵 |

## 5단계 실행 프레임워크
1. 입력 수집 (재무제표 3개년·업종·오너 정보)
2. 6트랙 병렬 분석
3. 4자관점 교차 검증 (법인·주주·과세관청·금융기관)
4. 절세 실행 우선순위 10개 산출
5. 산출물 패키징 (PPT·Excel·DOCX)

## 출력 형식
- 기본: PPT (.pptx, 네이비 #000080 표준)
- 요청 시: Excel 시뮬레이터 / DOCX 보고서 / PDF

## 품질 기준 (QUALITY_GATE)
- 수치 계산 자체검증 오류 0건
- 세법 조문 근거 명시 (조세특례제한법 우선)
- 4자관점 모두 반영
- 단정적 전문가 언어 (면책문구 금지)
