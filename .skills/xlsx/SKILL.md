# xlsx

## 역할
openpyxl 기반 Excel 시뮬레이터·분석 도구 생성 스킬.
네이비 표준 서식, 수식·조건부서식·차트 자동 구성.

## 트리거
- "Excel 만들어", "시뮬레이터 생성", "xlsx 출력"
- "가지급금 시뮬", "배당 급여 계산", "세금 계산기"

## 표준 서식

| 요소 | 값 |
|---|---|
| 헤더 배경 | 000080 (네이비) |
| 헤더 폰트 | 흰색, Bold, 10pt, Calibri |
| 데이터 폰트 | 1E293B, 9pt, Calibri |
| 짝수행 배경 | F8FAFC |
| 구분선 색상 | D1D9E6 |
| 수식 강조 | B45309 (앰버) |

## 다중 시트 구성 원칙
- Sheet1: 입력 파라미터 (노란색 입력셀)
- Sheet2: 계산 엔진 (수식 집중)
- Sheet3: 시나리오 비교 (A/B/C 옵션)
- Sheet4: 법령 근거 (조문·세율)
- Sheet5: 요약 대시보드 (KPI 카드 + 차트)

## 품질 기준
- 수식 오류 (#REF!, #DIV/0! 등) 0건
- 입력·계산·출력 셀 색상 구분
- 수식 자가검증 시트 포함
- 세율·공제 한도 최신화 (2026년 귀속 기준)
