# docx

## 역할
python-docx 기반 Word 보고서 생성 스킬.
네이비 표준 서식, 챕터·표·목차 자동 구성.

## 트리거
- "Word 보고서", "docx 생성", "보고서 작성"
- "계약서", "법무서류", "종합역량문서"

## 표준 서식

| 요소 | 값 |
|---|---|
| 제목 폰트 | Malgun Gothic, 22pt Bold, 네이비 000080 |
| 본문 폰트 | Malgun Gothic, 9~10pt |
| 표 헤더 | 네이비 배경 000080, 흰색 텍스트 |
| 여백 | 상하 1.8cm, 좌우 2.0cm |
| 표 구분선 | D1D9E6 |

## 챕터 구조 템플릿
1. 표제부 (기관명·제목·날짜·담당자)
2. 개요 (배경·목적·범위)
3. 본론 (챕터별 내용 + 표)
4. 결론·권고사항
5. 부록 (법령 근거·수식 검증)

## 품질 기준
- 수식 계산 자체검증 (오류 0건)
- 법령 조문 번호 명시
- 표 헤더 셀 배경색 매번 새 객체 (재사용 금지)
