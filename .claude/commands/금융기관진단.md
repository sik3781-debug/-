# /금융기관진단

## 단일 책임
법인 재무구조 기준 금융기관 신용등급 영향 요인을 진단하고 대출가용성·담보 전략을 제시한다.

## 트리거
- "금융기관 진단", "신용등급 진단", "대출 전략"
- "담보 전략", "부채비율 개선"

## 진단 5대 지표

| 지표 | 기준 | 위험 신호 |
|---|---|---|
| 부채비율 | 200% 이하 | 400% 초과 |
| 유동비율 | 100% 이상 | 70% 미만 |
| 이자보상비율 | 2배 이상 | 1배 미만 |
| 자기자본비율 | 30% 이상 | 15% 미만 |
| 영업현금흐름 | 양(+) 지속 | 2년 연속 음(-) |

## 대출 가용성 분석
- 기존 대출 만기·이자율·LTV 현황
- 추가 담보 여력 (부동산·보증 한도)
- 신용보증기금·기술보증기금 활용 가능성
- 정책자금 매칭 가능 여부

## 신용등급 영향 요인
- 재무비율 (40%)
- 업력·대표이사 신용 (20%)
- 산업 리스크 (20%)
- 기타 (20%)

## 입력
- 최근 2개년 재무제표
- 기존 차입금 현황
- 담보 자산 목록
- 업종·매출 추세

## 출력 형식
- 진단 보고서 (5대 지표 + 개선 로드맵)
- 금융기관별 대출 가능 금액 시뮬

## 자가검증
- 재무비율 계산 자체검증
- 4자관점 중 금융기관 관점 강조
- 개선 시 신용등급 상승 효과 계량화
