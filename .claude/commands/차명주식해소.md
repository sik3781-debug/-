# /차명주식해소

## 단일 책임
차명주식 실태 파악 → 실명전환 방안별 세부담 시뮬레이션 → 최적 해소 경로 제시.

## 트리거
- "차명주식 해소", "명의신탁 정리", "차명주식 실명전환"
- "주식 명의 변경", "주주명부 정리"

## 법적 근거
- 금융실명제법: 비실명 자산 실명전환 의무
- 상증세법 제45조의2: 명의신탁 증여의제 (증여세 과세)
- 시효: 명의신탁 사실 인지일부터 15년 이후 취득분 → 시효 쟁점

## 해소 방안 비교

| 방안 | 방법 | 세금 | 리스크 |
|---|---|---|---|
| A | 실명전환 (매매 형식) | 양도소득세 | 저가 거래 시 증여세 추징 |
| B | 증여로 실명전환 | 증여세 | 명의신탁 증여의제 중복 주의 |
| C | 자기주식 취득 후 소각 | 의제배당 | 법인 자금 필요 |
| D | 유상감자 | 의제배당 | 균등감자 요건 |
| E | 시효 완성 후 주주권 주장 | 최소화 가능 | 증빙 요건 충족 필수 |

## 입력
- 차명주식 내역 (주주명·주수·취득일·취득가)
- 실제 소유자 정보
- 현재 비상장주식 평가액
- 명의신탁 경위·증빙

## 출력 형식
- 방안별 세부담 비교 Excel
- 실명전환 실행 체크리스트

## 자가검증
- 양도세·증여세 계산 자체검증
- 명의신탁 증여의제 적용 여부 확인
- 세무조사 리스크 등급 평가
