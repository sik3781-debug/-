# /법인재무종합분석

## 단일 책임
재무제표 전 항목을 분석하여 법인세 절세 여지·손금 누락·과세 리스크를 종합 진단한다.

## 트리거
- "법인세 줄이는 방법", "법인세 절세", "법인 재무 분석"
- "세금 줄이기", "손금 누락 점검"

## 입력
- 최근 3개년 재무제표 (손익계산서·재무상태표)
- 업종 코드 및 주요 사업 내용
- 대표이사 급여 및 임원 구성
- 접대비·감가상각·퇴직급여충당금 현황
- 세무조사 이력 여부

## 분석 항목 (6트랙)

| 트랙 | 내용 |
|---|---|
| T1 법인세 과세표준 | 세무조정 항목 검토 |
| T2 손금 최적화 | 누락 손금·과다 익금 항목 |
| T3 세액공제·감면 | 미활용 공제 항목 발굴 |
| T4 임원 보수 | 급여·퇴직금 적정성 |
| T5 접대비·복리후생비 | 한도 초과·적용 오류 |
| T6 세무조사 리스크 | 고위험 항목 목록 |

## 출력 형식
- 과세표준 구간별 세부담 분석
- 손금 누락 항목 및 추가 손금산입 가능 금액
- 세액공제·감면 미활용 항목 점검
- 과세 리스크 항목 목록 (세무조사 가능성)
- 절세 실행 우선순위 10개 항목

## 자가검증
- 계산 정확성 확인
- 법령 근거 명시 (법인세법 우선)
- 4자관점 (법인·주주·과세관청·금융기관) 모두 반영
- 2026년 귀속 세법 기준 적용
