# /마스터컨설팅

## 단일 책임
법인·주주·과세관청·금융기관 4자관점을 통합한 종합 컨설팅 패키지를 1회 실행으로 산출한다.

## 트리거
- "종합 컨설팅", "전체 분석", "마스터 컨설팅"
- "법인 전체 진단", "종합 절세 전략"

## 입력
- 법인명·업종·사업연도
- 최근 3개년 재무제표
- 대표이사 급여·지분·가족 현황
- 주요 이슈 (가지급금·비상장주식·가업승계 등)

## 실행 순서
1. corp-financial-analysis 스킬 → 6트랙 분석
2. 4자관점 교차 분석
3. 절세 우선순위 10개 도출
4. 시나리오 A/B/C 비교
5. 실행 로드맵 (단기·중기·장기)

## 출력 형식
- PPT 10슬라이드 (네이비 표준)
- Excel 시뮬레이터
- 요약 1페이지 (경영자용)

## 자가검증
- 계산 오류 0건
- 4자관점 모두 반영
- 세법 근거 명시
- 단정적 전문가 언어
