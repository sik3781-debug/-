# /특허패키지

## 단일 책임
특허권 활용 법인 절세·가지급금 해소·기술이전 수익 최적화 패키지를 설계한다.

## 트리거
- "특허 절세", "특허 패키지", "기술이전"
- "연구개발비 공제", "특허권 현물출자"

## 특허 활용 절세 4가지 트랙

**트랙1: 연구개발비 세액공제 (조세특례제한법 제10조)**
- 중소기업: 연구개발비 × 25% 세액공제
- 일반기업: 연구개발비 증가분 × 50% (기본 + 추가)

**트랙2: 기술이전 소득 감면 (조세특례제한법 제12조)**
- 특허권·기술 이전 소득 → 50% 과세 감면 (중소기업 전용)

**트랙3: 특허 현물출자 가지급금 상계**
- 특허 현물평가 → 법인 자본금 납입 → 가지급금 상계
- 요건: 특허 가치평가 + 변리사 확인

**트랙4: 특허 담보 정책자금 활용**
- IP 담보 대출 (기술신용보증기금 등)
- 특허 담보 인정 범위: 등록 특허·실용신안

## 입력
- 보유 특허 목록 (특허번호·등록일·기술분야)
- 연구개발비 집행 현황
- 가지급금 잔액 (특허 현물출자 상계 시)
- 기술이전 희망 여부

## 출력 형식
- 트랙별 세부담 절감 효과 Excel
- 실행 체크리스트

## 자가검증
- 세액공제율 최신 적용 확인
- 특허 현물평가 요건 충족 여부
- 이중혜택 제한 규정 확인
