# /가업승계시뮬

## 단일 책임
가업상속공제·사전증여·자녀법인 설계 등 가업승계 시나리오별 세부담을 시뮬레이션하여 최적 경로를 제시한다.

## 트리거
- "가업승계", "가업상속공제", "경영권 이전"
- "자녀에게 주식", "2세 승계"

## 주요 세제 파라미터

**가업상속공제 (상증세법 제18조의2)**
- 한도: 최대 600억
- 요건: 피상속인 10년 이상 경영 + 상속인 가업 종사

**증여재산공제 (상증세법 제53조)**
- 배우자: 6억 (10년 합산)
- 직계존비속 성년: 5천만
- 직계존비속 미성년: 2천만
- 혼인공제: +1억 / 출산공제: +1억 (합산 최대 1억)

## 시나리오 비교

| 시나리오 | 방법 | 특징 |
|---|---|---|
| A | 가업상속공제 | 600억 한도, 사후관리 7년 |
| B | 사전 주식 증여 | 증여세·평가액 관리 중요 |
| C | 자녀법인 설립 후 사업 이전 | 법인세 이중과세 주의 |
| D | 가족법인 전환 | 지배구조 재설계 |
| E | A+B 복합 전략 | 중간 지분 분산 |

## 입력
- 기업 현황 (매출·순이익·자산·업종)
- 오너 연령·지분율·가족 현황
- 비상장주식 평가액
- 희망 승계 시기

## 출력 형식
- 시나리오 비교 Excel
- PPT 의사결정 보고서

## 자가검증
- 상속세·증여세 계산 자체검증
- 사후관리 요건 충족 여부 확인
- 4자관점 반영
